# DEV — Constitution Development Workspace

**Meta-development:** Using Omega principles to build Omega itself.

---

## What's Here

```
DEV/
├── CONTEXT_DEV.md       ← Our SESSION_CONTEXT (working memory)
├── CHANGES_DEV.md       ← Constitution changelog
├── IMPLEMENTATION_DEV.md ← Roadmap and ideas
├── evaluation.md        ← Framework evaluation notes
└── future-kits/         ← Incomplete kits (saas, api, automation)
```

---

## How We Work

1. **Start session:** Read CONTEXT_DEV.md
2. **Track changes:** Update CHANGES_DEV.md
3. **Plan work:** Use IMPLEMENTATION_DEV.md
4. **End session:** Update CONTEXT_DEV.md with summary

---

## Rules

- This folder is for **developing the framework**
- NOT for user projects (that's USER SPACE/)
- Changes here affect the constitution itself
- Commit to omega-constitution repo, not omega-store
